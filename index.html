Google Sheets — Apps Script
Integración para envío automático de calificaciones
URL del Web App (Apps Script)
📋 Instrucciones para configurar Google Apps Script:

1. Abre Google Sheets y ve a Extensiones → Apps Script
2. Pega el siguiente código en el editor:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([data.fecha, data.nombre, data.usuario, data.seccion, data.fase, data.aciertos, data.total, data.porcentaje + '%', data.tiempo]);
  return ContentService.createTextOutput('OK');
}

3. Guarda y haz clic en Implementar → Nueva implementación
4. Tipo: App web, acceso: Cualquier usuario
5. Copia la URL y pégala arriba

Las columnas del Sheet serán: Fecha | Nombre | Usuario | Sección | Fase | Aciertos | Total | % Nota | Tiempo
